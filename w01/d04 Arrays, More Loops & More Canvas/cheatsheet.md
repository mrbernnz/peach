#### For loops

```js
// prints 0 through 9
for(var i=0; i<10; i++) {
  console.log(i);
}
```

#### Arrays

```js
//Make a new empty array
var a = [];

//Make a new array with 3 elements
var b = [5,20,1];

//Sets the first element of b to 5
b[0] = 5;

//Adds a new element to an array
b.push(20);

//merge the contents of two arrays
a.concat(b);
```
