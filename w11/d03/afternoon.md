# Homework
Add Backbone Views to your Newspaper app to create a Single Page App.

You will need a view for the collection of Articles, as well as each individual article.
You can choose to render the show action if you want.
You will need to be able to create new articles and update articles.

You can work in pairs if you want.

**Bonus:** Use Bootstrap and add some styling.  Try to add some modals and play around with some of the features at http://getbootstrap.com/javascript/
